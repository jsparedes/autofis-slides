Pontos Negativos


Nos Sistemas Fuzzy Evolucionários

  • É o seu alto custo computacional e o grande número de parâmetros adicionados em decorrência do uso de um Algoritmo Evolucionário.


Nos Sistemas Fuzzy Evolutivos

  • Ao não presumir uma alocação inicial e fixa para as funções de pertinência, torna as regras dificilmente interpretáveis.

1.2 Objetivos


Objetivo Primário

  • Foi a proposição de um novo sintetizador automático genérico de SIF para Classificação.


Objetivos Secundários

  • Analisar as características do modelo proposto AutoFIS
  • Comparar os resultados do modelo AutoFIS com SFEs.

1.3 Modelo AutoFIS

Procura-se:

  • Geração automática de SIF para Classificação
  • Sem uso de Algoritmos Evolucionários

Vantagens:

  • Mais determinístico
  • Procura-se a redução do tempo computacional

Disvantagens:

  • Modelo com muitos parâmetros.
  • Se a base de dados é grande, a base pode ser muito custosa.

4. Conclusões

Conclusões


  • Foi gerada uma nova metodologia para a geração automática de Sistemas de Inferência Fuzzy.


  • As bases de regras geradas foram competitivas sem uso de algum Algoritmo Evolucionário, sua compactação foi apresentada na comparação com os outros Algoritmos Evolucionários.


  • Nos experimentos obteve o segundo melhor resultado em termos de acurácia no Grupo 1 e o primeiro lugar no Grupo 2.

5. Trabalhos Futuros

Trabalhos Futuros

  • Gerar premisas maiores, usando principios do algoritmo Wang & Mendel.
  • Extensão do modelo com uso aletoriedad na estrutura dos classificadores, como por exemplo em Random Forest para gerar diversidade de classificadores.
